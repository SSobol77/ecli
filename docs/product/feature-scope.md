# Feature Scope

## Core Scope

- Terminal text editing and navigation
- Syntax highlighting
- Undo/redo history
- Search and replace
- File operations

## Integrated Scope

- Git status and operations
- Lint diagnostics (Python Ruff + optional DevOps linter path)
- AI provider panel and responses
- Panel-based UI extensions (file browser, git panel, AI panel)

## Out-of-Scope (Current)

- GUI desktop editor parity
- Fully stable plugin ecosystem contract (planned area)
- Guaranteed enterprise-grade release support SLA
